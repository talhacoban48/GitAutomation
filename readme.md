# GitAutomation

**Git PR Automation Tool** – Hızlı ve kolay bir şekilde Git PR payload'ları oluşturmanıza yardımcı olur.  
Bu web uygulaması tamamen **tarayıcı tabanlı**dır, ek bir kurulum gerektirmez.

---

## Ne İşe Yarar?

GitAutomation, PR oluştururken sık kullandığınız bilgileri tek bir sayfada toplar ve hazır hale getirir:

- Hangi branch’e PR açılacağını seçmek
- Issue numarasını girmek
- Opsiyonel branch ismi eklemek
- Commit mesajı oluşturmak
- İlgili repository’yi seçmek veya eklemek
- Tüm bu bilgileri hızlıca kopyalayabilmek

---

## Kullanım Talimatları

1. **Branch Type Seçin**
   - Açılır menüden PR’nin hangi branch’e merge edileceğini seçin.
   - Yeni bir branch tipi ekleyebilir veya mevcutları silebilirsiniz.  
   - Silmek için yanındaki `-` ikonuna tıklayın ve onaylayın.

2. **Issue Number Girin**
   - PR ile ilgili issue numarasını girin.

3. **Branch Name (Opsiyonel)**
   - PR için özel bir branch adı eklemek isterseniz buraya yazabilirsiniz.

4. **Commit Message**
   - Yapılan değişiklikleri açıklayan commit mesajınızı girin.

5. **Issue Repository Seçin**
   - Mevcut repository’lerden birini seçin veya yeni bir repository ekleyin.
   - Repository’leri de tıpkı branch tipleri gibi silebilirsiniz, silme onaylıdır.

6. **PR Payload Oluştur**
   - “Create PR Payload” butonuna tıklayın.
   - Sayfanın alt kısmında oluşturulan branch name, commit message ve PR title görüntülenecektir.

7. **Kopyalama**
   - Her bir alanın yanında bulunan **Copy** butonuna tıklayarak bilgileri panoya kopyalayabilirsiniz.
